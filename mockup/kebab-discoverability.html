Mockup: Kebab Discoverability — 4 Variants

Compare 4 strategi biar user notice icon ✨ di header tanpa bikin bingung. Click tombol "Replay" di tiap panel untuk lihat animasi dari awal.

Varian A
First-time Header Balloon

Tooltip balloon kecil nempel di icon ✨ selama 6 detik pas user pertama kali buka chatbox. Cuma muncul sekali per session (track di localStorage). Dismiss otomatis atau pas user click icon.

Varian D
First-click Reveal Toast

User pertama kali click icon ✨ → muncul mini-toast "Tepat! Aku punya mode spesial 🎉" + menu kebab kebuka. Cuma works kalau user klik buta. Track localStorage biar ga muncul lagi.

Varian E
Welcome Chip "Mode"

Tambah chip "Mode" di welcome screen (sebaris dgn Topik & Coaching). Click → langsung buka menu kebab. Onboarding via entry point yg udah ada, zero visual cost di header.

Combo: Balloon + Welcome Chip

Multi-channel: user baru liat welcome chip "Mode" (E) → click langsung ke menu. Plus header balloon muncul 6s (A) sebagai backup kalo user skip welcome. Track di localStorage, semua sekali per session.

Cara compare:
• A — auto-show 6s pas chatbox kebuka, click "Replay" buat liat anim. Click "Dismiss" buat dismiss manual.
• D — click "Click icon" buat simulasi user click icon ✨ pertama kali (toast muncul).
• E — click "Click chip" buat simulasi user click chip "Mode" di welcome.
• A + E — click "Replay full flow" buat liat welcome chip muncul + balloon auto-show 1.5s kemudian.

Catatan: Semua variant first-time only (track via localStorage `ava_kebab_hint_seen`), jadi user ga terganggu di session berikutnya.
